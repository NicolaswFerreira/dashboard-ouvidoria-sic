# Dashboard-ouvidoria-sic
Dashboard em Excel para análise de manifestações de Ouvidoria/SIC com base simulada e indicadores de gestão pública.

Dashboard de Ouvidoria/SIC — Base Simulada

Este projeto apresenta um dashboard desenvolvido em Excel para análise de manifestações de Ouvidoria e pedidos de acesso à informação, utilizando uma base simulada e sem dados pessoais.

O objetivo é demonstrar a aplicação de conceitos de gestão pública, análise de dados e visualização de indicadores em um contexto administrativo.

## Objetivo do projeto

Construir um painel gerencial capaz de apoiar o acompanhamento de demandas, prazos, unidades responsáveis e gargalos operacionais em fluxos de Ouvidoria/SIC.

## Indicadores analisados

- Total de manifestações
- Distribuição por status
- Prazo médio de resposta
- Taxa de resposta no prazo
- Evolução mensal das demandas
- Demandas por unidade
- Demandas por tipo
- Assuntos mais recorrentes
- Gargalos por unidade

## Ferramentas utilizadas

- Microsoft Excel
- Tabelas dinâmicas
- Gráficos dinâmicos
- Segmentações de dados
- Tratamento e padronização de base

Cuidados com dados

Este projeto não utiliza dados reais de cidadãos, protocolos oficiais ou informações internas sensíveis. A base foi simulada/anonimizada para fins de portfólio.
