# Dashboard-ouvidoria-sic
Dashboard em Excel para análise de manifestações de Ouvidoria/SIC com base simulada e indicadores de gestão pública.

Dashboard de Ouvidoria/SIC — Base Simulada

Este projeto apresenta um dashboard desenvolvido em Excel para análise de manifestações de Ouvidoria e pedidos de acesso à informação, utilizando uma base simulada e sem dados pessoais.

O objetivo é demonstrar a aplicação de conceitos de gestão pública, análise de dados e visualização de indicadores em um contexto administrativo.

## Objetivo do projeto

Construir um painel gerencial capaz de apoiar o acompanhamento de demandas, prazos, unidades responsáveis e gargalos operacionais em fluxos de Ouvidoria/SIC.

## Indicadores analisados

- Total de manifestações
- Distribuição por status
- Prazo médio de resposta
- Taxa de resposta no prazo
- Evolução mensal das demandas
- Demandas por unidade
- Demandas por tipo
- Assuntos mais recorrentes
- Gargalos por unidade

## Ferramentas utilizadas

- Microsoft Excel
- Tabelas dinâmicas
- Gráficos dinâmicos
- Segmentações de dados
- Tratamento e padronização de base

# Metodologia

## 1. Base de dados

A base utilizada neste projeto é simulada e foi estruturada para representar um fluxo administrativo de manifestações de Ouvidoria/SIC.

Foram considerados campos como:

- data de entrada;
- data de resposta;
- status;
- unidade responsável;
- tipo de manifestação;
- assunto;
- prazo;
- situação da resposta.

## 2. Tratamento dos dados

Foram realizadas etapas de:

- padronização de nomes de unidades;
- classificação de status;
- cálculo de prazo de resposta;
- identificação de respostas dentro e fora do prazo;
- agrupamento por mês;
- preparação da base para tabelas dinâmicas.

## 3. Indicadores

Os indicadores foram definidos com foco em acompanhamento gerencial, permitindo visualizar volume de demandas, cumprimento de prazos e possíveis gargalos por unidade.

## 4. Limitações

Por se tratar de base simulada, os dados não representam números oficiais. O objetivo do projeto é demonstrar a estrutura analítica e a lógica de construção do dashboard.
